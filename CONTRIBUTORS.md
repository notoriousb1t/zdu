# Zelda: Dungeons Unseen

**ZDU** | A randomizer for the Legend of Zelda

## Project Structure

* **zdu_ui (Rust/Iced)**: Configures randomization options and manages multiplayer sessions.
* **zdu_connector (Lua)**: Hooks directly into emulators for bidirectional state manipulation.
* **zdu_lib (Rust)**: Loads and updates the base game using patches
